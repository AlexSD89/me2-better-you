# 合作伙伴资源

## 目录用途
本目录用于管理LaunchX的各类合作伙伴资源，包括客户、供应商、媒体、投资和战略合作伙伴。

## 目录结构
```
/03_合作伙伴list/
├── 客户资源/                # 现有和潜在客户资料
├── 供应商资源/              # AI工具和服务供应商
├── 媒体资源/                # 媒体平台和KOL资源
├── 投资合作/                # 投资机构和标的
└── 战略合作/                # 战略合作伙伴
```

## 内容组织方式

### 客户资源/
存放现有和潜在客户的详细资料，包括：
- 客户基本信息
- 合作历史
- 需求文档
- 联系人信息

### 供应商资源/
存放AI工具和服务供应商的资料，包括：
- 供应商基本信息
- 产品/服务介绍
- 合作条款
- 联系人信息

### 媒体资源/
存放媒体平台和KOL资源的资料，包括：
- 媒体平台信息
- KOL资源列表
- 合作方式和价格
- 联系人信息

### 投资合作/
存放投资机构和投资标的的资料，包括：
- 投资机构信息
- 投资标的资料
- 投资条款
- 联系人信息

### 战略合作/
存放战略合作伙伴的资料，包括：
- 合作伙伴基本信息
- 合作内容和范围
- 合作协议
- 联系人信息

## 文件命名规范
- 合作伙伴名称_合作类型_日期.扩展名
- 例如：北京骏开科技_技术合作_20250419.pptx
