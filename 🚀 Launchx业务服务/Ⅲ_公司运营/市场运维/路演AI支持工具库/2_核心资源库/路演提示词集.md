# AI路演提示词集

## 内容创作提示词

### 路演大纲生成

```
请为一场关于[主题]的20分钟路演创建详细大纲。目标受众是[受众描述]，路演目标是[目标描述]。

请提供：
1. 整体结构（建议3-5个主要部分）
2. 每部分的关键内容要点（3-5点）
3. 建议的时间分配
4. 开场吸引注意的方式
5. 结束部分的行动号召

请确保内容具有连贯性，并突出[核心差异点/价值/创新点]。
```

### 演讲稿生成

```
请根据以下大纲为我创建一篇20分钟的路演演讲稿。演讲主题是[主题]，面向[受众类型]。

演讲大纲：
1. [第一部分标题]
   - [要点1]
   - [要点2]
2. [第二部分标题]
   - [要点1]
   - [要点2]
3. [第三部分标题]
   - [要点1]
   - [要点2]

请注意：
- 开场需要引人入胜，可以使用[故事/数据/问题/引言]
- 语言风格应该[正式/轻松/专业/富有激情]
- 每个部分之间要有自然过渡
- 结尾需要有明确的行动号召
- 增加适当的互动元素或修辞问题
- 整体演讲稿应保持20分钟左右（约2500-3000字）
```

### 问答准备

```
请帮我准备一个关于[主题]的路演问答环节准备材料。假设观众是[受众类型]，他们最可能提出以下几类问题：

1. 关于[技术/产品/服务]的实现细节
2. 关于市场竞争和差异化的问题
3. 关于成本和投资回报的问题
4. 关于未来发展规划的问题
5. 可能的质疑或挑战

请为每类问题提供3-5个最可能被问到的具体问题，并为每个问题提供简明扼要（100字以内）但内容充实的回答。回答应该专业、自信，并包含具体数据或例子。
```

## 互动设计提示词

### 实时AI演示设计

```
请帮我设计一个5分钟的实时AI演示环节，用于在[主题]路演中展示AI技术的能力。

演示应该：
1. 能在现场使用ChatGPT或Midjourney等工具实时完成
2. 允许观众提供输入或参与
3. 产生令人印象深刻的结果
4. 能直观地展示AI的价值
5. 有明确的开始和结束流程

请提供：
- 演示的详细步骤说明
- 需要准备的材料或工具
- 引导观众参与的脚本
- 应对可能问题的备用方案
- 1-2个预设的示例提示词，确保演示成功
```

### 观众互动环节设计

```
请为一场关于[主题]的路演设计一个10分钟的观众互动环节，使用AI工具增强参与度。

互动环节应：
- 能够快速收集观众输入（关键词、问题、选择等）
- 使用AI实时处理这些输入
- 生成有趣或有价值的输出
- 直观展示AI如何增强人类工作
- 适合[观众规模]人的场合

请详细说明：
1. 互动的具体形式和流程
2. 所需的技术和工具
3. 主持人引导语的示例
4. 如何展示处理结果
5. 可能遇到的问题和备用方案
```

## 虚拟主持人提示词

### 主持人角色设计

```
请帮我创建一位AI虚拟主持人的角色设定，用于[主题]路演。

基本信息：
- 名称：（建议简单、国际化）
- 性别表达：[男性/女性/中性]
- 年龄感：[年龄范围]岁
- 专业背景：与[行业/领域]相关
- 外观特点：[简要描述]

性格与表达风格：
- 主要性格特点（3-5个特点）
- 语言风格（正式程度、专业术语使用、是否幽默等）
- 说话特点（语速、重点强调方式等）

主持人的背景故事：
（100字以内的简短背景，解释为什么这个虚拟主持人适合此次路演）
```

### 完整主持脚本

```
请为我的路演创建一个完整的AI虚拟主持人脚本。路演主题是[主题]，目标观众是[受众描述]。

虚拟主持人是[名称]，[性别]，[年龄]岁，具有[专业背景]，语言风格[风格描述]。

请为以下各环节创建脚本，每段控制在100字以内：

1. 开场白：问候观众，介绍自己和活动
2. 活动议程介绍：简述本次路演将包含的[X]个环节
3. 演讲者介绍：介绍主讲人[姓名]及其背景（假设主讲人是[职位]的[姓名]）
4. 第一环节到第二环节的转场词
5. 提示观众互动的引导词
6. 第二环节到第三环节的转场词
7. 结束语：总结活动，感谢观众，说明后续安排

请在脚本中添加表情和停顿标记，如(微笑)、(停顿)、(手势)等，使主持更自然。
```

## 邀请函生成提示词

### 路演邀请函内容

```
请为一场关于[主题]的路演创建一份专业且吸引人的邀请函内容。

活动信息：
- 主题：[详细主题]
- 日期与时间：[日期和时间]
- 地点：[详细地址]
- 主要嘉宾：[嘉宾信息]

目标受众：[受众描述]

请提供：
1. 一个简短但吸引人的标题
2. 开场段落（说明活动意义和独特价值）
3. 活动亮点（3-5点关键吸引点）
4. 嘉宾介绍（简洁突出核心吸引力）
5. 面向受众的价值主张（他们能获得什么）
6. 清晰的RSVP信息和截止日期
7. 简短的结束语

语调应专业但不乏热情，突出活动的独特价值和创新性。
```

## 实时演示提示词

### AI内容生成演示

```
我正在进行一场关于AI创新应用的路演，现在是实时演示环节。请根据观众提供的以下关键词，在90秒内生成一个创新的[产品/服务/解决方案]概念：

关键词：[观众提供的关键词]
行业：[相关行业]
目标用户：[目标用户]

请提供：
1. 概念名称（简短、有记忆点）
2. 一句话描述（清晰说明是什么）
3. 解决的核心问题（为什么重要）
4. 3个关键功能或特点（如何工作）
5. 潜在商业模式（如何创造价值）
6. 实施所需的核心技术（技术可行性）

要求简洁、创新、具有实用价值，并展示AI如何解决实际问题。
```

### 实时视觉创作演示

```
我正在进行AI路演的实时演示环节。观众提供了以下概念，请创建一个视觉化的表现：

概念：[观众提供的概念]
行业应用：[应用场景]
风格要求：专业、现代、吸引人
颜色偏好：[颜色偏好]

请创建一个能直观表达这个概念的视觉设计，适合用于商业演示。设计应简洁明了，能在几秒内让人理解核心思想。
```
