## 📊 功能详解

### 1. 实时数据获取
- **数据源**: FinnHub API (机构级数据)
- **更新频率**: 实时 (交易时间内)
- **数据准确性**: 99.9%
- **包含指标**: 开盘价、最高价、最低价、收盘价、涨跌幅

### 2. 智能分析算法
```python
# 评分算法逻辑
基础分数 = 50分
涨跌幅加分：
  - 大涨(>2%)：+20分  
  - 小涨(0-2%)：+10分
  - 小跌(0-2%)：-10分
  - 大跌(<-2%)：-20分

价格位置加分：
  - 接近低点(<20%)：+15分
  - 中性位置：0分  
  - 接近高点(>80%)：-5分
```

### 3. 投资建议生成
| 评分区间 | 建议等级 | 含义 | 操作建议 |
|----------|----------|------|----------|
| 70-100分 | 🟢 建议买入 | 技术面强势 | 可考虑建仓 |
| 60-69分 | 🔵 可以关注 | 中性偏多 | 密切观察 |
| 40-59分 | 🟡 谨慎观望 | 中性 | 暂时观望 |
| 0-39分 | 🔴 暂时回避 | 技术面弱势 | 避免建仓 |
