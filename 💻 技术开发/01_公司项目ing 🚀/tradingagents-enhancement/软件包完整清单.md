## 🆘 常见问题解决

### ❓ Q1: 运行时提示"No module named 'requests'"
**解决方案**:
```bash
pip3 install requests
# 或
pip install requests
```

### ❓ Q2: API连接失败
**解决方案**:
1. 检查网络连接
2. 确认防火墙设置
3. 验证API密钥有效性

### ❓ Q3: Python版本问题
**解决方案**:
```bash
# 检查Python版本
python3 --version

# 如果版本过低，请升级至3.6+
```

### ❓ Q4: 权限问题 (macOS/Linux)
**解决方案**:
```bash
chmod +x run.sh
chmod +x *.py
```

### ❓ Q5: Windows系统路径问题
**解决方案**:
- 确保文件路径不包含中文
- 使用CMD或PowerShell运行
- 避免使用文件夹中的空格
