## 🎨 色彩系统精细化设计

### 基于60-30-10原则的智链配色方案
```yaml
主色调体系(60%_界面主体):
  智链深蓝: '#1a365d'
    - 使用场景: 主导航、页面背景、大面积区域
    - 色彩心理: 专业、可信赖、科技感
    - 对比度检查: 与白色文字对比度4.8:1 (WCAG AA标准)
  
  浅灰背景: '#f7fafc' 
    - 使用场景: 卡片背景、输入框背景
    - 视觉效果: 柔和、不刺眼、长时间使用舒适

次色调体系(30%_功能区域):
  智链科技蓝: '#3182ce'
    - 使用场景: CTA按钮、重要链接、进度条
    - 悬停状态: '#2c5aa0' (加深15%)
    - 激活状态: '#2a69ac' (加深12%)
  
  功能区灰: '#e2e8f0'
    - 使用场景: 分割线、边框、次要按钮
    - 悬停效果: '#cbd5e0' (加深一级)

强调色体系(10%_关键元素):
  成功绿: '#38a169' 
    - 使用场景: 成功状态、完成标识、正向反馈
    - 可访问性: 与白色对比度4.6:1
  
  预警橙: '#ed8936'
    - 使用场景: 警告信息、待处理状态
    - 注意: 不单独依赖颜色，配合图标使用
  
  错误红: '#e53e3e'
    - 使用场景: 错误状态、删除操作、风险提示
    - 配套图标: 感叹号、X符号确保可访问性
  
  高亮金: '#d69e2e'
    - 使用场景: 重要数据突出、VIP标识
    - 使用限制: 仅用于关键信息突出显示
```

## ✍️ 字体系统与层级设计

### 基于无障碍原则的字体选择
```yaml
中文字体栈:
  primary: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei UI', 'Microsoft YaHei'
  理由: 无障碍性优先，对老年人和视觉障碍者更友好的无衬线字体
  
英文字体栈:
  primary: 'Inter', 'system-ui', '-apple-system', 'BlinkMacSystemFont', 'Segoe UI'
  理由: Inter字体在企业应用中清晰度和可读性表现优秀
  
代码/数据字体:
  monospace: 'JetBrains Mono', 'Fira Code', 'SF Mono', 'Monaco', 'Menlo'
  特色: 支持连字符，提升代码阅读体验
  使用场景: API文档、配置代码、数据表格
```

### 字体大小层级系统
```css
/* 基于1.25倍等比数列的字体层级 */
:root {
  --font-size-xs: 0.75rem;    /* 12px - 辅助信息 */
  --font-size-sm: 0.875rem;   /* 14px - 次要内容 */
  --font-size-base: 1rem;     /* 16px - 正文基准 */
  --font-size-lg: 1.125rem;   /* 18px - 重要正文 */
  --font-size-xl: 1.25rem;    /* 20px - 小标题 */
  --font-size-2xl: 1.5rem;    /* 24px - 中标题 */
  --font-size-3xl: 1.875rem;  /* 30px - 大标题 */
  --font-size-4xl: 2.25rem;   /* 36px - 主标题 */
}

/* 字重层级 */
.font-light { font-weight: 300; }    /* 辅助信息 */
.font-normal { font-weight: 400; }   /* 正文内容 */
.font-medium { font-weight: 500; }   /* 强调内容 */
.font-semibold { font-weight: 600; } /* 小标题 */
.font-bold { font-weight: 700; }     /* 重要标题 */

/* 行高优化 */
.leading-tight { line-height: 1.25; }   /* 标题使用 */
.leading-normal { line-height: 1.5; }   /* 正文使用 */
.leading-relaxed { line-height: 1.625; } /* 长文本使用 */
```

## ⚡ 性能优化与加载体验

### 渐进式加载设计
```css
/* 骨架屏加载效果 */
.skeleton {
  background: linear-gradient(90deg, 
    #f0f0f0 25%, 
    #e0e0e0 50%,

  );
  background-size: 200% 100%;
  animation: loading 1.5s infinite;
}

@keyframes loading {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}

.skeleton-text {
  height: 16px;
  border-radius: 4px;
  margin-bottom: 8px;
}

.skeleton-text:last-child {
  width: 60%;
}

/* 懒加载图片渐显 */
.lazy-image {
  opacity: 0;
  transition: opacity 0.5s ease;
}

.lazy-image.loaded {
  opacity: 1;
}

/* 内容逐步显现 */
.fade-in {
  opacity: 0;
  transform: translateY(20px);
  animation: fadeInUp 0.6s ease forwards;
}

.fade-in:nth-child(1) { animation-delay: 0.1s; }
.fade-in:nth-child(2) { animation-delay: 0.2s; }
```
